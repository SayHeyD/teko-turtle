# Cutting Board Drawer Optimizer

## Setup

### Dev

```shell
make install-dev
```

### Prod

```shell
make install
```
   
## Commands

* Lint
    ```shell
    make lint
    ```
* Fix codestyle (does not fix all linting errors automatically)
    ```shell
    make lint-fix
    ```
* Running tests
    ```shell
    make test
    ```
