# Cutting Board Drawer Optimizer

A CLI program for calculating what cutting board sizes fit into which drawer sizes and 
optimizing the distribution of these cutting boards into the drawers.

> UI made with [Textual](https://textual.textualize.io/)

## Setup

Requires python 3.14 or newer

```shell
make setup-venv
```

### Dev

```shell
make install-dev
```

### Prod

```shell
make install
```
   
## Commands

* Lint
    ```shell
    make lint
    ```
* Fix codestyle (does not fix all linting errors automatically)
    ```shell
    make lint-fix
    ```
* Running tests
    ```shell
    make test
    ```
